# Game Manager (GM)

A web-based game planning and stage management tool for Unity projects.

## Overview
Game Manager is a personal project built to organize game development data such as worlds, stages, goals, rewards, and progress status.

## Purpose
While working on Unity game projects, I wanted a simple tool to manage stage planning and development flow more efficiently.

## Tech Stack
- React
- Vite
- Tailwind CSS
- Supabase
- Cloudflare Pages
